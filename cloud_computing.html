Cloud Computing

2025 Cloud Computing

What is cloud computing?

Cloud computing delivers on-demand computing resources (servers, storage, databases, networking, software) over the internet, enabling scalable and flexible IT services without local infrastructure.

Is cloud computing a new technology per se? In other words, what are the unique technology features of cloud computing, if any?

Cloud computing builds on virtualization, distributed systems, and automation. Its unique features include resource pooling, rapid elasticity, on-demand self-service, and measured service.

What are the three major cloud service models?

  • Infrastructure as a Service (IaaS): Provides virtualized computing resources over the internet.
  • Platform as a Service (PaaS): Offers hardware and software tools for application development without managing underlying infrastructure.
  • Software as a Service (SaaS): Delivers software applications over the internet on a subscription basis.

Name at least 3 real-world domains where the application of cloud computing was (or can be) very enabling and effective.

  • Healthcare (e.g., scalable data analytics and telemedicine platforms)
  • Finance (e.g., high‑performance trading platforms and regulatory compliance solutions)
  • Education (e.g., virtual learning environments and resource sharing)
  • Entertainment (e.g., streaming services and online gaming backends)
  • Retail (e.g., e‑commerce platforms and inventory management)

What is the economic/business model of cloud computing?

The cloud business model is pay-as-you-go (utility pricing), where customers are billed based on resource usage (compute time, storage, bandwidth), reducing capital expenditure and aligning costs with demand.
